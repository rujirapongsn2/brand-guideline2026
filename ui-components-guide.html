States Reference

แนวทางการใช้สีสำหรับ Component States ต่างๆ

Component States — Quick Reference

State Light Mode Dark Mode CSS Example
Default Primary color 100% Primary color 100% background: #F3903F;
Hover Brightness -10% หรือ Shadow เพิ่ม Brightness +5% หรือ Glow เพิ่ม filter: brightness(0.9);
Active/Pressed Brightness -15% Brightness -5% filter: brightness(0.85);
Focus Primary color + Ring shadow Primary color + Glow ring box-shadow: 0 0 0 3px rgba(color, 0.25);
Disabled Opacity 50% + cursor not-allowed Opacity 40% + cursor not-allowed opacity: 0.5; cursor: not-allowed;
Loading Opacity 70% + spinner Opacity 70% + spinner opacity: 0.7; pointer-events: none;

Pro Tips for States

  • Hover Transition: ใช้ transition: all 0.2s ease; สำหรับ smooth animation
  • Focus Ring: สำหรับ Accessibility ควรมี focus ring ที่ชัดเจน contrast ratio 3:1 ขึ้นไป
  • Disabled States: ไม่ควรใช้สีเทาอย่างเดียว ควร maintain brand color แต่ลด opacity
  • Dark Mode Hover: ใน Dark Mode ควรเพิ่ม brightness แทนที่จะลด เพราะ UI จะสว่างขึ้นเมื่อ interact

Opacity Reference (สำหรับ Background)

  • Light Mode Background: ใช้ opacity 10-15% ของ Primary color
  • Dark Mode Background: ใช้ opacity 15-20% ของ Primary color
  • Light Mode Border: ใช้ opacity 30% ของ Primary color
  • Dark Mode Border: ใช้ opacity 30-40% ของ Primary color
  • Shadow (Light): rgba(color, 0.2) สำหรับ subtle, rgba(color, 0.3) สำหรับ prominent
  • Glow (Dark): rgba(color, 0.3) สำหรับ subtle, rgba(color, 0.4) สำหรับ prominent
